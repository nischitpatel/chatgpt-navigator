A lightweight Chrome extension that helps you navigate long ChatGPT conversations without endless scrolling.
The extension collects all **questions you ask** and displays them in a **collapsible sidebar**.
Clicking a question instantly scrolls ChatGPT to that part of the conversation.

---

## Features

* Automatically detects and stores your ChatGPT questions
* Clean, vertical sidebar with a salt-and-pepper theme
* Toggle button attached to the right edge of the screen
* Click a question to jump to that exact response
* Search through past questions
